# DurstBot

Miscellaneous Discord bot for the UOB AMPS Discord server. 

## Technologies

Durstbot is hosted on [Replit](https://replit.com/) and built with Python 3.8.

Libraries:
- flask 2.0.1
- discord.py 1.7.3
- PyNaCl 1.4.0
- pandas 1.4.2
- Levenshtein 0.18.1
- thefuzz 0.19.0
